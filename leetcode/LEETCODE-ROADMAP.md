## How to Use This List

### 1. Week 1-4 (Foundation): Focus on Easy & Medium
- Start with: Arrays, Hash Tables, Two Pointers, Strings
- Do 5-7 problems per week (each in 3 languages)
- Build pattern recognition

### 2. Week 5-8 (Core Patterns): Medium Problems
- Focus on: Trees, Graphs, Binary Search, Sliding Window
- Do 5-7 problems per week
- Start recognizing when to use which pattern

### 3. Week 9-12 (Advanced): Medium & Hard
- Focus on: DP, Backtracking, Advanced Graphs
- Do 5-7 problems per week
- Optimize solutions

### 4. Week 13-16 (Interview Prep): Company-Specific
- Review all patterns
- Do mock interviews
- Solve company-tagged problems
- Practice explaining solutions
