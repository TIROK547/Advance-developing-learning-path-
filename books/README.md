# Books & Reading Materials

This directory contains PDFs and notes for technical books in the learning roadmap.

## Books List

### Algorithms & Data Structures
- **Grokking Algorithms** by Aditya Bhargava
  - Visual approach to algorithms
  - Great for understanding core concepts

### System Design & Architecture
- **Designing Data-Intensive Applications** by Martin Kleppmann
  - Essential for backend engineers
  - Deep dive into distributed systems, databases, and data processing

- **Clean Architecture** by Robert C. Martin
  - Software architecture principles
  - Building maintainable systems

### Software Craftsmanship
- **The Pragmatic Programmer** by David Thomas & Andrew Hunt
  - Timeless programming wisdom
  - Professional development practices
